It is better not to have every teacher write their own style
because it prevents the styles from clashing when navigating from one page to another.

    Tags and classes that may be used
  • ol - to order the exams
  • li - to list the students
  • table, tr, td, th- to create the table for the timetable and the grade table
  • h1-h6 - to display the heading of subjects etc
  • background color - for timetable etc
  • a - to have links that will take the student to another section
  • span - to set a color a grade that needs to be drop
  • body - to create the body of the syllabus etc
  • p- to write the text or paragraph
  • style - to style the table etc
